## 1. 概述

项目默认使用**内置 3D 演示地图**即可运行全部功能。要使用真实高德 3D 地图（真实道路、地名、建筑、POI），需申请高德 Web 端 JS API Key 并配置前端环境变量。

---

## 2. 申请高德 Key

1. 访问 [高德开放平台](https://lbs.amap.com/) 注册开发者账号。
2. 进入"应用管理 -> 我的应用"，创建新应用。
3. 添加 Key，服务平台选择 **"Web 端 (JS API)"**。
4. 获取 **Key** 和 **安全密钥**（JS Code）。

---

## 3. 创建前端环境文件

将 `frontend/.env.example` 复制为 `frontend/.env`，然后填写以下变量：

```env
VITE_API_BASE_URL=http://localhost:3000
VITE_AMAP_KEY=你的高德Web端Key
VITE_AMAP_SECURITY_JS_CODE=你的安全密钥
```

> 模板文件为 `frontend/.env.example`，目标文件为 `frontend/.env`。请勿直接修改 `.env.example`。

---

## 4. 重启前端

修改 `frontend/.env` 后**必须重启前端**，Vite 仅在启动时读取环境变量。运行：

```cmd
cmd /c npm run dev:frontend
```

或使用项目根目录的启动脚本。如果用 `npm run dev` 同时启动了前后端，也需重启整个 dev 进程。

---

## 5. 地图效果

配置成功后，"动态地图"页将使用以下特性：

| 特性 | 说明 |
|------|------|
| 底图 | 高德 JS API 2.0 真实瓦片底图 |
| 视角 | 3D 俯视（zoom=11，含建筑/地名/POI） |
| 路线 | AMap.Driving 真实道路路径，沿路网贴合 |
| 车辆标记 | SVG chevron 导航箭头，每条线路独立颜色 |
| 多线路监控 | 10 条线路同屏同步运行（140ms 帧率） |
| 高亮交互 | 点击线路高亮，遥测面板显示实时速度/人数/满载率 |

## 6. 验证地图是否正常加载

配置完成后，按以下步骤验证：

1. 打开浏览器**开发者工具 (F12)**，切换到 **Console** 面板。
2. 访问"动态地图"页面。
3. 观察控制台输出：
   - 正常：无 AMap 相关报错，地图瓦片正常加载（Network 面板中可见 `is.autonavi.com` 请求）。
   - 异常：出现 `AMap is not defined` 或 `Key 无效` 等错误。
4. 地图区域应显示**真实道路、建筑轮廓、地名标签和 POI 标记**。
5. 使用鼠标拖拽可旋转/倾斜地图（3D 模式）。

---

## 7. 自动回退机制

如果未配置 Key、Key 无效或 Key 过期，系统**不会报错中断**，而是自动回退到内置 3D 演示地图。回退行为：

- 底图回退为内置轻量地图。
- 线路轨迹、车辆 Marker、动画等功能**全部保留**。
- 保证课堂教学或答辩展示不受 Key 配置问题影响。

## 8. 新增依赖 AMap 的功能

以下 V2.0 新增功能**依赖高德 API**，需配置 Key 后方可生效：

| 功能 | 所需 API | 无 Key 时行为 |
|------|----------|---------------|
| 真实道路路线 | Driving API（驾车路线规划） | 回退备用路径/直线 polyline |
| 全线路箭头同屏监控 | AMap Marker + setAngle 旋转 | 回退 SVG 地图箭头 |
| 实时遥测面板 | AMap Marker 坐标更新 | 回退静态总览面板 |
| 3D 底图/建筑/POI | AMap Map 3D 渲染 | 回退内置 SVG 地图 |

> 这些功能在无 Key 模式下均可降级运行，不影响系统整体演示。

详见 [功能说明](./功能说明.md) 了解完整的 V2.0 功能列表。

## 9. 常见问题排查

| 问题 | 可能原因 | 解决方法 |
|------|----------|----------|
| 页面报错 `AMap is not defined` | JS API 脚本未加载 | 检查 `VITE_AMAP_KEY` 是否正确，确认网络可访问高德 CDN |
| 地图显示灰色/空白 | Key 无效或未配置安全密钥 | 检查 Key 是否启用"Web 端 (JS API)"服务，确认安全密钥正确 |
| 地图瓦片加载失败 (403) | Key 未绑定域名或安全密钥错误 | 在高德控制台检查 Key 的域名白名单设置 |
| Driving API 路线规划失败 | 未开通 Web 服务 API 或配额不足 | 在高德控制台开通"Web 服务 API"，检查每日配额 |
| 3D 视角无法旋转 | Key 版本不支持 3D | 确认使用 JS API 2.0 版本 |
| 配置后仍显示内置地图 | 未重启前端 | 停止 dev server 后重新运行 `npm run dev:frontend` |
| `frontend/.env` 不生效 | Vite 缓存 | 删除 `node_modules/.vite` 目录后重启 |
