# 数据字典

## bus_routes 公交线路表

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| id | INT | 线路主键 |
| number | VARCHAR | 线路编号，如 44路 |
| name | VARCHAR | 平台展示名称 |
| start_stop | VARCHAR | 起点站 |
| end_stop | VARCHAR | 终点站 |
| operation_time | VARCHAR | 运营时间 |
| fare | VARCHAR | 票价 |
| type | VARCHAR | 线路类型 |
| color | VARCHAR | 地图线路颜色 |

## bus_stops 公交站点表

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| id | INT | 站点主键 |
| name | VARCHAR | 站点名称 |
| lng | DECIMAL | 经度 |
| lat | DECIMAL | 纬度 |
| district | VARCHAR | 所属行政区 |

## tourist_spots 旅游景点表

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| id | INT | 景点主键 |
| name | VARCHAR | 景点名称 |
| lng / lat | DECIMAL | 经纬度 |
| category | VARCHAR | 景点类型 |
| district | VARCHAR | 所属行政区 |
| rating | DECIMAL | 推荐评分 |
| heat | INT | 热度指数 |
| intro | VARCHAR | 简介 |

## route_stops 线路站点关联表

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| route_id | INT | 线路 ID |
| stop_id | INT | 站点 ID |
| sequence | INT | 站点在线路中的顺序 |

## route_statistics 线路运营统计表

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| route_id | INT | 线路 ID |
| stat_date | DATE | 统计日期 |
| passenger_flow | INT | 仿真客流量 |
| punctuality | DECIMAL | 准点率 |
| congestion | INT | 拥挤度 |
| heat | INT | 热度值 |

## collection_samples 公交运行采集样本表

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| id | INT | 样本主键 |
| route_id | INT | 线路 ID |
| collected_at | DATETIME | 采集时间 |
| speed | DECIMAL | 车辆速度，单位 km/h |
| passenger_count | INT | 车内人数 |
| load_rate | INT | 满载率百分比 |
| source | VARCHAR | 数据来源，如 GPS 轨迹终端、车载客流传感器 |
