# 昆明景点真实图片存放规则

这个文件夹用于存放你下载的真实景点图片。页面会优先读取这里的图片；如果没有找到，就自动使用 `frontend/public/spot-images/` 里的 SVG 占位图。

## 命名规则

统一使用小写英文文件名，推荐 `.jpg` 格式：

| 景点 | 文件名 |
| --- | --- |
| 翠湖公园 | `cuihu.jpg` |
| 云南大学 | `yunda.jpg` |
| 南屏街 | `nanping.jpg` |
| 金马碧鸡坊 | `jinma.jpg` |
| 大观公园 | `daguan.jpg` |
| 海埂公园 | `haigeng.jpg` |
| 云南民族村 | `minzu.jpg` |
| 滇池大坝 | `dianchi.jpg` |
| 世博园 | `shibo.jpg` |
| 官渡古镇 | `guandu.jpg` |
| 斗南花市 | `dounan.jpg` |
| 西山风景区 | `xishan.jpg` |

## 图片建议

- 比例：16:9 或接近 16:9
- 推荐尺寸：`1280x720`、`1600x900` 或更高
- 文件大小：建议每张小于 1MB，网页加载更快
- 格式：优先 `.jpg`，也可以用 `.png`

## 注意

当前代码默认优先找 `.jpg`。如果你一定要用 `.png`，需要把对应路径改成 `.png`，或者告诉我我来改成自动兼容多格式。
